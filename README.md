# Torneo Ex Alumnos

Base tecnica de Sprint 1 para la plataforma web del torneo.

## Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- React Router
- TanStack Query
- Supabase
- React Hook Form + Zod

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`

## Variables de entorno
Copiar `.env.example` a `.env` y completar:
- `VITE_APP_NAME`
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `VITE_ADMIN_BYPASS`
